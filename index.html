I'm Front End Developer

Olá, meu nome é Enzo da Costa Xavier, sou formado técnico em Análise e Desenvolvimento de Sistemas pela ETEC Dra. Ruth Cardoso e atualmente sou estudante autodidata.

Meu primeiro contato com a programação foi em 2019 quando iniciei o curso de ADS e que acabou despertando um interesse maior por tecnologia.

Tenho conhecimento nas linguagens HTML, CSS, JS e Bootstrap e estou iniciando os estudos na biblioteca React JS para ampliar meus conhecimentos. Também possuo conhecimentos em algumas linguagens back end: PHP e MySQL.

Tecnologias

HTML5

HTML é uma linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.

CSS3

CSS é utilizado para formatar elementos escritos em uma linguagem de marcação como o HTML.

JS

Javascript é uma linguagem de programação voltada para web, de fácil execução e que pode ser rodada a partir de um navegador de internet.

SASS

O Sass é um pré-processador é como se fosse um CSS com super poderes cheio de funcionalidades que depois de compilar gera o CSS.

Bootstrap

Trata-se de um framework voltado para o desenvolvimento web. Ele é composto por uma série de códigos que se relacionam entre si para disponibilizar funcionalidades específicas.

React JS

O React é uma biblioteca JavaScript de código aberto com foco em criar interfaces de usuário em páginas web. É mantido pelo Facebook, Instagram, outras empresas e uma comunidade de desenvolvedores individuais.
